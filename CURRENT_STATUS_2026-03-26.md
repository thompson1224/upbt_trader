# Upbit AI Trader 현재 상태 정리

기준 시각: 2026-03-26 KST

## 완료

- 자동매매 기본 루프
- 주문/체결 동기화
- 실계좌 잔고 동기화
- SL/TP 실행
- 감사로그 저장/조회
- 프론트 운영 UI와 진단 흐름
- 성과 대시보드 및 코인 상세 분석 화면
- 제외 코인 관리, 제외 사유 메모, 제외/복귀 감사로그
- `hold` 저장 및 `hold/sell` 미실행 사유 가시화
- 실거래 성과 재구성 1차 정확도 개선
  - `market` 단일 상태머신에서 `FIFO lot` 기준으로 변경
- 코인 상세 전환 품질 조회 경량화
  - 전역 성과 재호출 제거
- 오늘 운영 요약 조회형 1차
  - 일일 손익, 리스크 거절, 주문 실패, 열린 포지션, 제외 코인 현황을 대시보드에서 바로 확인 가능
- 일일 운영 리포트 저장형 1차
  - 날짜별 snapshot 저장
  - 최근 운영 이력 비교 카드 추가
- 전략 검증 자동화 1차
  - 일일 snapshot에 `final score`, `시간대`, `취약 코인`, `리스크 거절 사유` 분석 축 저장
  - 대시보드에 `최근 7일 변화` 비교 카드 추가
- 운영 안정성 보강
  - `auto-trade` fail-closed
  - signal claim 기반 중복 실행 완화
  - orphaned `approved` 신호 복구

## 주의

- 실거래 성과 표본 수가 아직 적음
- 전략 매도는 개선됐지만 장기 데이터는 더 필요함
- 추천/제외 기준은 운영 중 계속 튜닝 필요
- 성과/분석 API는 개선됐지만 거래 수가 더 늘면 추가 최적화 여지 있음

## 미완

- 전략 수익성 검증 완료
- 장기 24/7 무인운영 확정
- `risk_service`, `backtest_service` 실제 서비스화
- 워크포워드 / 아웃오브샘플 검증 체계화
- 장기 비교용 일일 운영 리포트 시각화 확장

## 현재 판단

- 시스템은 `실거래 가능한 운영형 프로토타입` 수준까지 올라옴
- 가장 큰 남은 과제는 `전략이 실제로 돈을 버느냐`를 충분한 표본으로 증명하는 것
